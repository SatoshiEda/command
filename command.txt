git コマンド一覧

■gitの管理下に指定する
 % git init
 Initialized empty Git repository in {パス}.git/
上記のメッセージが出て「.git」ディレクトリが作成されればOK

■ファイルをインデックスする
% git add .

■ステージングされた内容を確認する
% git ls-files --stage
もしくは
% git status

■コミット
% git commit -m "最初のコミット"

■正常にコミットされているか確認
% git status
↓ステージングエリアにコミット対象のファイルなし
nothing to commit, working tree clean
↓まだステージングエリアには追加されていない状態
modified:   sample.txt

■リモートリポジトリを指定
% git remote add origin https://github.com/ユーザー名/リポジトリ名.git

■接続先のリモートリポジトリを確認
% git remote -v
origin  https://github.com/ユーザー名/ブランチ名.git (fetch)
origin  https://github.com/ユーザー名/ブランチ名.git (push)

■リモートリポジトリにpushする
% git push -u origin main
以降
% git push
でorigin mainにpushできる

■「test-branch」を追加する
% git checkout -b test-branch

■現在のブランチを確認
% git branch
  main
* test-branch

■作成したブランチにpushする
% git push -u origin test-branch
以降
% git push
でorigin test-branchにpushできる
